# Hospital
Modern responsive website for Sai Keerthi Dental Clinic in Kukatpally, Hyderabad. The site highlights dental treatments, doctor profile, patient reviews, clinic details, and contact information. Built with HTML, CSS, and JavaScript to provide a clean, professional platform for patients to explore services and book consultations

Features

Responsive clinic website design

Doctor profile and experience section

Dental treatment and services showcase

Patient testimonials and ratings

Clinic location, timings, and contact information

Clean and professional healthcare UI
